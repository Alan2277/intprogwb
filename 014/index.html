Atividade 14 - Transpârencias, Sombras, Gradientes.

RGBA

No RGBA a tranparência é localiza somente em em determinado elemento, assim se coloco para ter transparência em uma box s vai ser exibida naquela box sem afetar os outros elementos que estão dentro dela.
Sintaxe de utilização: 'background: rgb(255,255,255,x)'
x: Vária de 0 à 1 para o nível de transparência.

OPACITY

A opacidade é diferente do RGBA quando colocar a tag opacidade ela também influência nos demais elementos, desse modo, a tag pai que está com opacidade determina que seus filhos também vão ter aquela mesma opacidade.
Sintaxe: 'opacity: x'
x: Vária de 0 à 1 para o nível de transparência.
